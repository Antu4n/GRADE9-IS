Activity Introduction

Everyday electric shocks and static cling are signs that tiny particles are on the move. In this activity, you will explore real-life events caused by static electricity and choose the correct particle responsible. Read each scenario, select your answer, and learn from the feedback. Let us go!

Instructions

  1. Read each real-life scenario carefully.
  2. Select the particle you believe is responsible.
  3. Review the instant feedback to learn why.
  4. Hint: Electrons are the only atomic particles that can move and create electric charges.
